CSS Cheat-List

Selectors

Type

Just like its name suggests, the type selector matches the type of the element in the HTML document. The element type is p, which comes from the HTML <p> element. The type selector does not include the angle brackets. Since element types are often referred to by their opening tag name, the type selector is sometimes referred to as the tag name or element selector. 

 
p {
  color: green;
}

Universal

The universal selector selects all elements of any type. The universal selector uses the * character in the same place where you specified the type selector in a ruleset, like so: 

 
* { 
  font-family: Verdana;
 }

Class

Classes are HTML attributes used to select one or more elements for CSS styling or JavaScript purposes.The class attribute is commonly used for assigning styles to one or more elements.

    
   


<p class='brand'>Sole Shoe Company</p>


.brand {
 color: red;
} How code looks like in the CSS

Multiple Classes

We can add multiple classes to an HTML element’s class attribute by separating them with a space. This enables us to mix and match CSS classes to create many unique styles without writing a custom class for every style combination needed.

    
    


   
.green {
  color: green;
}
.bold {
  font-weight: bold;
}
         
<h1 class='green bold'>...</h1>

ID

Oftentimes it’s important to select a single element with CSS to give it its own unique style. If an HTML element needs to be styled uniquely, we can give it an ID using the id attribute.

 
  


<h1 id='large-title'>..</h1>
 
css
 
 #large-title {
    font-size:30px;
  }

Attribute

Attributes can be selected similarly to types, classes, and IDs. 

      
      
      
      
                 
                 
                 
             <img src='/images/seasons/cold/winter.jpg'>
             <img src='/images/seasons/warm/summer.jpg'> - in HTML
       In CSS
          img[src*='winter'] {
            height: 50px;
          }
       
          img[src*='summer'] {
           height: 100px;
          }

Pseudo-class

These are all examples of pseudo-class selectors in action! In fact, :focus, :visited, :disabled, and :active are all pseudo-classes. Factors such as user interaction, site navigation, and position in the document tree can all give elements a different state with pseudo-class. A pseudo-class can be attached to any selector. It is always written as a colon : followed by a name. 

p:hover {
 background-color: lime;
}

Specificity

Specificity is the order by which the browser decides which CSS styles will be displayed. A best practice in CSS is to style elements while using the lowest degree of specificity so that if an element needs a new style, it is easy to override. IDs are the most specific selector in CSS, followed by classes, and finally, type. 




                             
                      
                      
                      
                      
      <h1 class='headline'>Breaking News</h1>
      
      h1 {
       color: red;
      }    
 
     .headline {
      color: firebrick;
     }

Chaining

When writing CSS rules, it’s possible to require an HTML element to have two or more CSS selectors at the same time. 

 h1.special {
 }

Descendant Combinator

The descendant combinator — typically represented by a single space (" ") character — combines two selectors such that elements matched by the second selector are selected if they have an ancestor (parent, parent's parent, parent's parent's parent, etc.) element matching the first selector. 

    
  
   <ul class='main-list'>
      <li> ... </li> 
      <li> ... <li>
      <li> ... </li>
   </ul>

Command Line Cheat-List

Working with Terminal

Command Name Description
ls List The command line looks at the directory you are in,and then “lists” all the files and directories inside of it.
pwd Print working directory It outputs the name of the directory you are currently in, called the working directory.
cd Change directory Just as you would click on a folder in Windows Explorer or Finder, cd switches you into the directory you specify. In other words, cd changes the working directory.
cd .. Change directory back on one step The argument .. stands for the directory above the current working directory.
cd / Change directory into two steps To move across multiple directories with a single command, we can provide cd a relative path. The relative path is a forward slash (/) separated list of all the directories leading to the goal directory.
mkdir Make directoryt It takes in a directory name as an argument and then creates a new directory in the current working directory.
touch Touch The touch command creates a new file inside the working directory.It takes in a filename as an argument and then creates an empty file with that name in the current working directory.
